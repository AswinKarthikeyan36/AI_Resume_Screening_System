# AI_Resume_Screening_System
The AI Resume Screening System is a Python-based application that helps recruiters quickly screen candidate resumes based on required skills. The system compares the candidate's resume with predefined job skills and calculates a matching score.
The application stores candidate information in a database and analyzes resumes by extracting skills and matching them with the required job skills. Based on the match, the system identifies suitable candidates for a particular role.

This project demonstrates the implementation of basic AI concepts such as keyword-based resume analysis and automated candidate screening.

Features

Add candidate details and resume information

Store candidate data in a SQL database

Extract skills from resumes

Match candidate skills with required job skills

Automatically identify suitable candidates

Simple and user-friendly implementation

How It Works

Candidate details are entered into the system.

The information is stored in the database.

The system analyzes the skills provided in the resume.

It compares candidate skills with the required job skills.

Candidates with matching skills are identified as suitable for the job.
